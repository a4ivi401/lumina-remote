# LuminaRemote

LuminaRemote — це ультрашвидкий, легкий та безпечний додаток для віддаленого доступу, оптимізований для нестабільних мереж та локальних підключень.

## Основні можливості
- **Zero-Trust Безпека**: Жодних паролів. Аутентифікація здійснюється за допомогою 12-символьної сід-фрази з використанням X25519 та Argon2id.
- **P2P Архітектура**: Прямі з'єднання через протокол QUIC.
- **Адаптивний QoS**: Динамічна зміна бітрейту, FPS та роздільної здатності залежно від стану мережі (RTT, втрати пакетів).
- **Оптимізація для LAN**: Миттєве виявлення через mDNS та трансляція з мінімальною затримкою.
- **Кросплатформність**: Розроблено на Rust та Tauri для підтримки Windows, macOS та Linux.
