# LuminaRemote

LuminaRemote — это ультрабыстрое, легковесное и безопасное приложение для удаленного доступа, оптимизированное для нестабильных сетей и локальных подключений.

## Основные возможности
- **Zero-Trust Безопасность**: Никаких паролей. Аутентификация по 12-символьной сид-фразе с использованием X25519 и Argon2id.
- **P2P Архитектура**: Прямые соединения через протокол QUIC.
- **Адаптивный QoS**: Динамическое изменение битрейта, FPS и разрешения в зависимости от состояния сети (RTT, потери пакетов).
- **Оптимизация для LAN**: Мгновенное обнаружение через mDNS и трансляция с минимальной задержкой.
- **Кроссплатформенность**: Разработано на Rust и Tauri для поддержки Windows, macOS и Linux.
